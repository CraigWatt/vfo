## vfo Installation and Quick Start

### Supported platforms (this release)
- macOS 13+ (Apple Silicon ARM64 and Intel x64)
- Linux/Windows installers are not yet published

### Runtime requirements
- Required:
  - `ffmpeg` (includes `ffprobe`)
  - `mkvmerge` (from `mkvtoolnix`)
- Optional:
  - `dovi_tool` (recommended for Dolby Vision workflows)
  - `ffmpeg` with `libvmaf` filter (only if using `QUALITY_CHECK_INCLUDE_VMAF=true`)

vfo now checks required dependencies before pipeline execution and clearly reports what is missing.
Use `vfo status` or `vfo status-json` to inspect component/stage readiness before running the pipeline.
`vfo mezzanine-clean` provides optional mezzanine folder/filename hygiene with probe-based recommendations.

### macOS (recommended: signed package)
1. Download `vfo_macOS_installer-v*.*.*.pkg` from this release's assets.
2. If macOS blocks first launch, right-click the package and choose **Open**.
3. Complete the installer.
4. Verify installation:
   - `vfo --version`
   - `vfo doctor`

Installed paths:
- Binary: `/usr/local/bin/vfo`
- Action scripts: `/usr/local/bin/transcode_*_profile.sh`
- Config directory: `/usr/local/bin/vfo_conf_folder`
- Config file: `/usr/local/bin/vfo_conf_folder/vfo_config.conf`
- Stock presets: `/usr/local/bin/vfo_conf_folder/presets/`

### macOS (advanced: static build zip)
1. Download `static_build.zip` from assets and unzip it.
2. Move `static_build/bin/vfo` and all `static_build/bin/transcode_*_profile.sh` scripts to `/usr/local/bin`.
3. Create config directory and copy the default config:
   - `sudo mkdir -p /usr/local/bin/vfo_conf_folder`
   - `sudo cp static_build/vfo_conf_folder/vfo_config.conf /usr/local/bin/vfo_conf_folder/vfo_config.conf`
   - `sudo cp -R static_build/vfo_conf_folder/presets /usr/local/bin/vfo_conf_folder/`
4. Verify installation:
   - `vfo --version`
   - `vfo doctor`

### First-run workflow
1. `vfo wizard` to generate or update config interactively.
   - wizard now supports stock preset pack multi-select (`balanced_open_audio`, `device_targets_open_audio`, `netflixy_main_subtitle_intent`)
2. `vfo show` to review active config values.
3. `vfo doctor` to validate paths and required tools.
4. `vfo status` (or `vfo status-json`) to confirm engine readiness.
5. `vfo mezzanine-clean` to audit (or apply) mezzanine naming/structure hygiene.
6. `vfo run` to execute the default pipeline (including optional quality scoring when enabled).

### Multi-location storage routing
- `MEZZANINE_LOCATIONS`, `SOURCE_LOCATIONS`, and `<PROFILE>_LOCATIONS` support semicolon-separated paths.
- `MEZZANINE_LOCATION_MAX_USAGE_PCT`, `SOURCE_LOCATION_MAX_USAGE_PCT`, and `<PROFILE>_LOCATION_MAX_USAGE_PCT` set per-path usage caps.
- vfo routes writes by projected utilization and free-space reserve, then retries on alternate locations when a destination fails.

### Optional mezzanine hygiene controls
- `MEZZANINE_CLEAN_ENABLED`: enable/disable automatic mezzanine hygiene in `vfo run`.
- `MEZZANINE_CLEAN_APPLY_CHANGES`: `false` = audit only, `true` = perform rename/move.
- `MEZZANINE_CLEAN_APPEND_MEDIA_TAGS`: append tags like `UHD/HDR10/HEVC/AAC` to filenames.
- `MEZZANINE_CLEAN_STRICT_QUALITY_GATE`: fail when mezzanine warnings are detected.

### Optional post-profile quality scoring controls
- `QUALITY_CHECK_ENABLED`: enable/disable quality scoring stage in `vfo run`.
- `QUALITY_CHECK_INCLUDE_VMAF`: include VMAF when FFmpeg has `libvmaf`; otherwise PSNR/SSIM only.
- `QUALITY_CHECK_STRICT_GATE`: fail run on threshold failures/missing references/metric errors.
- `QUALITY_CHECK_REFERENCE_LAYER`: `auto|source|mezzanine` reference selection.
- `QUALITY_CHECK_MIN_PSNR`: minimum PSNR threshold (`0` disables threshold).
- `QUALITY_CHECK_MIN_SSIM`: minimum SSIM threshold (`0` disables threshold).
- `QUALITY_CHECK_MIN_VMAF`: minimum VMAF threshold (`0` disables threshold).
- `QUALITY_CHECK_MAX_FILES_PER_PROFILE`: sample cap per profile (`0` means all files).

### Troubleshooting
- `required command missing: ffmpeg/ffprobe`:
  - `brew install ffmpeg`
- `required command missing: mkvmerge`:
  - `brew install mkvtoolnix`
- `optional command missing: dovi_tool`:
  - `brew install dovi_tool`
- `could not create config directory` during `vfo wizard`:
  - run with elevated permissions, for example: `sudo vfo wizard`
- Gatekeeper warning on `.pkg`:
  - right-click installer -> **Open**
