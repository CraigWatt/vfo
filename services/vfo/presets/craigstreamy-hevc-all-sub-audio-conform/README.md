# craigstreamy-hevc-all-sub-audio-conform

Fixed-name `craigstreamy` stock pack for:

- HEVC video
- `all_sub_preserve + preserve` subtitle policy
- `audio_conform` delivery behavior

This pack keeps every subtitle stream that is present, while still conforming DTS-family and PCM-family audio into open-source Dolby-aligned delivery codecs when needed.
