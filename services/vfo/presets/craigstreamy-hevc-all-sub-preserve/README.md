# craigstreamy_hevc_all_sub_preserve

`craigstreamy_hevc_all_sub_preserve` is the explicit subtitle carry-over variant in the craigstreamy HEVC family.

It keeps:

- HEVC video outputs
- copied audio streams
- `all_sub_preserve + preserve` subtitle handling
- MKV whenever subtitle streams are selected for retention

Included profiles:

- `craigstreamy_hevc_all_sub_preserve_4k`
- `craigstreamy_hevc_all_sub_preserve_1080p`
- `craigstreamy_hevc_all_sub_preserve_legacy_subhd`
