# craigstreamy-hevc-smart-eng-sub-subtitle-convert-audio-conform

Fixed-name `craigstreamy` stock pack for:

- HEVC video
- `smart_eng_sub + subtitle_convert` subtitle policy
- `audio_conform` delivery behavior

This pack converts selected text subtitles into delivery-friendly subtitle text when MP4 remains viable, but still conforms DTS-family and PCM-family audio when needed.
If another constraint forces MKV, the selected subtitles are preserved rather than OCR-converted.
