# Demo Pack Replay

The standalone browser app now lives at [vfo-web-app/](vfo-web-app/).

It is a full static page, separate from the normal docs article layout, so you can open it directly on GitHub Pages without the documentation chrome around it.

The page is now driven by a dashboard contract that can switch between replay lanes, so the same route can show a live e2e-backed transcript or the fallback demo pack model.

Use this page if you want the explainer and a link into the app route:

- the left rail shows the mezzanine corpus
- the center lane shows the left-to-right workflow transcript for the selected lane
- the right rail shows the selected node output in a code-style inspector
- the lower band shows summary totals and stage counts

For the actual browser app, open [vfo-web-app/](vfo-web-app/).
