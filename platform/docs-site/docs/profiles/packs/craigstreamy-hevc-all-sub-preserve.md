# Craigstreamy HEVC All Sub Preserve Pack

This pack is the carry-over oriented subtitle variant in the `craigstreamy` HEVC family.

It keeps:

- HEVC video outputs
- copied audio streams
- `all_sub_preserve + preserve` subtitle handling
- MKV whenever subtitle streams are retained

Use it when broad subtitle retention matters more than the leaner `smart_eng_sub` viewing-intent posture.
