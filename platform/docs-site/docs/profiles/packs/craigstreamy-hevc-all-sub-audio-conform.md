# Craigstreamy HEVC All Sub Audio Conform Pack

This pack combines the broadest subtitle carry-over posture in the current `craigstreamy` family with the stricter `audio_conform` delivery strategy.

It keeps:

- HEVC video outputs
- `all_sub_preserve + preserve`
- every subtitle stream that packaging can safely carry

It adds:

- preserve-first audio for AAC and Dolby-family sources
- DTS-family and PCM-family conform into open-source Dolby-aligned delivery codecs when needed
- loudness normalization only on the audio streams that are actually transcoded
